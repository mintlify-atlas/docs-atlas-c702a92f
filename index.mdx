The NCBI C++ Toolkit is a large-scale, production-tested C++ library developed by the National Center for Biotechnology Information (NCBI). It provides everything you need to build bioinformatics applications: from a robust application framework and serialization infrastructure to the full BLAST sequence alignment engine and the Object Manager for working with biological sequence data.

## Getting started

<Steps>
  <Step title="Get the source code">
    Clone or download the public repository from GitHub:
    ```bash
    git clone https://github.com/ncbi/ncbi-cxx-toolkit-public.git
    cd ncbi-cxx-toolkit-public
    ```
  </Step>
  <Step title="Configure the build">
    Run the configure script to detect your platform and generate build files:
    ```bash
    ./cmake-configure --with-build-root=build
    ```
  </Step>
  <Step title="Build the toolkit">
    Compile the core libraries:
    ```bash
    cd build/build && make -j4
    ```
  </Step>
  <Step title="Write your first application">
    Subclass `CNcbiApplication`, implement `Run()`, and link against the toolkit libraries. See the [Quickstart](/quickstart) for a complete working example.
  </Step>
</Steps>

<Note>
The NCBI C++ Toolkit is a United States Government Work and is freely available for use by the public. It is used internally at NCBI to power tools including BLAST, Entrez, and the Sequence Viewer.
</Note>
